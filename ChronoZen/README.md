# ChronoZen — Application Windows (C# / WPF)

## Pour obtenir le fichier ChronoZen.exe

### Étape 1 — Installer le .NET 8 SDK (une seule fois, gratuit)
👉 https://dotnet.microsoft.com/download/dotnet/8.0  
Choisir : **SDK** → **Windows** → **x64**  
(~220 Mo, installation en 2 minutes)

### Étape 2 — Compiler
Double-cliquez sur **`COMPILER.bat`**  
→ Crée automatiquement `dist\ChronoZen.exe`

### Étape 3 — Lancer
Double-cliquez sur `dist\ChronoZen.exe`  
✅ Aucune dépendance, fonctionne sur n'importe quel Windows 10/11

---

## Fonctionnalités
- 🕐 Horloge analogique avec aiguilles temps réel
- ⏱ Timer principal réglable (MM:SS)
- 🎯 Aiguilles synchronisées sur le timer en cours
- 🔔 Notification Windows même en arrière-plan
- 💤 Snooze +5 minutes
- ☕ Timer de pause dédié
- ✨ Motifs décoratifs évolutifs (0→12+ timers/jour)
- 📊 Compteur journalier persisté dans %APPDATA%\ChronoZen\
